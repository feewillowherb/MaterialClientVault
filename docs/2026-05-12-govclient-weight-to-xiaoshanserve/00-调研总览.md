## 关键发现摘要

1. **稳定性检测极其简单**：基于 `System.Timers.Timer`（2 秒间隔），连续 5 次重量不再增大即视为稳定，无窗口分析、无范围计算。
2. **全链路同步依赖 SQLite + Timer 轮询**：本地写入 SQLite → 5 秒定时器轮询 → HTTP POST 到 XiaoShanServe → XiaoShanServe 再转发到政府平台。
3. **XiaoShanServe 是中转层**：它接收 GovClient 数据后存储到本地 SQLite，再通过后台服务转发到真正的政府 API（`http://191.12.15.58:8899/sapi/v1/inoutRecord/save`）。
4. **无进出场匹配**：GovClient 每次称重独立上报，不做进出配对和运单创建（与 MaterialClient 的核心差异）。
5. **代码质量差距明显**：WinForms 事件驱动、`Application.DoEvents()`、全局锁、无依赖注入、大量废弃代码。
