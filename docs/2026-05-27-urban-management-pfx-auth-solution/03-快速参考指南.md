## 快速实施清单

### 第一阶段：基础准备（1-2 周）

#### 1. 证书生成
```bash
# 生成 CA 证书
openssl req -x509 -newkey rsa:4096 -keyout ca-key.pem -out ca-cert.pem -days 365

# 生成服务器证书
openssl req -newkey rsa:2048 -keyout server-key.pem -out server-req.pem
openssl x509 -req -in server-req.pem -CA ca-cert.pem -CAkey ca-key.pem -CAcreateserial -out server-cert.pem -days 365

# 生成客户端 PFX 证书
openssl pkcs12 -export -out client.pfx -inkey client-key.pem -in client-cert.pem
```

#### 2. 项目配置
```json
// appsettings.json

```

// 双重验证服务
public class DualValidationService
{
    public async Task<bool> ValidateAsync(X509Certificate2 cert, string licenseNo)

        var certValid = ValidateCertificate(cert);

        return certValid && licenseValid;
    }
}

## 代码模板库

### 客户端模板

```csharp
// UrbanManagement.Client/Security/CertificateService.cs
public class CertificateService : ICertificateService
{
    private readonly X509Certificate2 _certificate;
    private readonly ILogger<CertificateService> _logger;

    public CertificateService(IConfiguration config, ILogger<CertificateService> logger)
    {
        _logger = logger;
        var path = config["Certificate:Path"];
        var password = config["Certificate:Password"];
        _certificate = new X509Certificate2(path, password);
    }

    public async Task<HttpClient> CreateAuthenticatedClientAsync()
    {
        var handler = new HttpClientHandler
        {
            ClientCertificates = { _certificate }
        };
        return new HttpClient(handler);
    }
}
```

// UrbanManagement.Server/Middleware/CertificateValidationMiddleware.cs
public class CertificateValidationMiddleware
{
    public async Task InvokeAsync(HttpContext context, ICertificateValidator validator)

        var cert = await context.Connection.GetClientCertificateAsync();
        if (!validator.Validate(cert))

}

## 常见问题快速解答

### Q1: PFX 证书与现有许可证号验证冲突吗？
**不冲突**。采用双重验证：
- PFX 证书：主要验证，高安全性
- 许可证号：辅助验证，保证兼容性

### Q2: 如何处理旧客户端不支持证书的问题？
**向后兼容策略**：
```csharp
if (certificate != null)
{
    // 新客户端：双重验证
    return await ValidateWithCertificate(certificate, licenseNo);
}
else
{
    // 旧客户端：仅许可证号验证
    return await ValidateWithLicenseOnly(licenseNo);
}
```

### Q3: 证书过期如何处理？
**自动续期机制**：
```csharp
public class CertificateRenewalService : BackgroundService
{
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            var cert = await _certManager.LoadCertificate();
            if ((cert.NotAfter - DateTime.Now).Days < 30)
            {
                await RenewCertificate();
            }
            await Task.Delay(TimeSpan.FromDays(1), stoppingToken);
        }
    }
}
```

### Q4: 性能影响有多大？
**性能优化措施**：
- 证书缓存：避免重复加载
- 连接池：复用 SSL 连接
- 会话恢复：减少握手次数

**预期性能**：
- 证书验证：< 50ms
- API 调用增加：< 100ms
- 总体影响：< 20%
